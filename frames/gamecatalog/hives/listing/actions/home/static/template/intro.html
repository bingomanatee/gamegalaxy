Game Galaxy

This is a visualization of the GiantBomb API using the Famo.us display system.

The first thing you need to do is to enter in a search term. This can be a partial or full game name, then click List. The GiantBomb API can be a bit slow, so give it a few seconds. Also, large datasets can take a few passes to load ... watch the load progress in the footer

Rolling over a game row changes the display on the right, showing the name, art and a button to view a description of the game. Clicking once on a row "locks" the rollover, allowing you to get to the callout description button without accidentally rolling over another game. Clicking again on the same game "Unlocks" the row.

Clicking the "Description" button on the callout displays a text description of the game.

Not all games listed have a description and images

Filtering the game list

The game list can be sorted by name or release date by cliking on a row header.

Once you have a list of games, clicking on the "◐" button opens a dialog showing all the platforms of all the games; click on a platform to reduce the list to only show games playable on that platform. Click the "X" by the Platform name in the header to view games for all platforms.

Viewing the games in a timeline

Once you have a dataset you can click or over over a list item to view detail on it in the right panel.

You can also flip between list and timeline views with the buttons in the upper right corner.
